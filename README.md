# QR Cranker
Easy, private and reliable QR codes. Simple front-end app to be deployed by github pages.

## Ease
- Input any compatible data
- Render a testable preview
- Customise appearance
- Export with one click

## Privacy
- No ads
- No database
- Nothing stored
- No data in URL params
- Necessary analytics only (e.g. traffic, nothing idengtifying)

## Reliability
- High durability
- Integrated safe area options
- Appearance defaults aligned to spec
- Alert if customisation breaks spec
